TodoList-zadání

Jednoduchá aplikace ToDo Manager, která umožňuje uživatelům vytvářet a spravovat úkoly. 
Každý úkol by měl obsahovat termín splnění a být přiřaditelný do různých kategorií (např. škola, soukromé, práce). 
Bude implementovat funkci štítků (tagů) pro lepší organizaci a bude mít vyhledávací panel pro rychlé nalezení úkolů podle názvu nebo štítků.
